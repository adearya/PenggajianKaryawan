### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/adearya/PenggajianKaryawan.git
   ```
2. Change directory
   ```sh
   cd PenggajianKaryawan/
   ```
3. If you want to recompile
   ```sh
   rm Main.class
   javac Main.java
   ```
## Usage
   ```sh
   java Main
   ```
